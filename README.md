# 📌 Local Job Board (Full Stack + Docker)

A full-stack Job Board web application built using **FastAPI (Backend)**, **Next.js (Frontend)**, and **PostgreSQL**, fully containerized using **Docker Compose** for easy setup and deployment.

---

## 🚀 Features

- Job listings creation and browsing
- REST API built with FastAPI
- PostgreSQL database integration
- Modern frontend using Next.js
- Full containerization using Docker
- Easy one-command setup using Docker Compose

---

## 🏗️ Tech Stack

### Frontend
- Next.js
- React

### Backend
- FastAPI
- SQLAlchemy
- Uvicorn

### Database
- PostgreSQL 15

### DevOps
- Docker
- Docker Compose
